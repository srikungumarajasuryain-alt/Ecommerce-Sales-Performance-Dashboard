# 📊 E-Commerce Sales Performance Dashboard

## 📌 Project Overview

This project is an end-to-end E-Commerce Sales Performance Dashboard built using SQL, Power BI, and Python. The dashboard helps analyze sales, customers, products, and overall business performance through interactive visualizations and KPIs.

The project demonstrates the complete analytics workflow from database creation to dashboard development.

---

## 🎯 Project Objectives

- Analyze overall sales performance
- Monitor customer behavior and segmentation
- Identify top-performing products
- Evaluate business performance across states
- Analyze payment methods and order status
- Build interactive dashboards for business decision-making

---
